## 概要
このリポジトリは、Rustで作ったパスワード作成ツールのリポジトリになります。

## ビルド
[こちら](https://dioxuslabs.com/learn/0.6/guide/tooling#setting-up-tooling)を参考にしてください。
```
git clone https://github.com/8bitTD/password_dioxus
cd password_dioxus
dx serve --release
```
